Ahmad Syafii, Universitas Negeri Semarang. Saya adalah anak kedua dari dua bersaudara. Umur saya 20 tahun. Sekarang saya berkuliah di Universitas Negeri Semarang, program studi Teknik Informatika, jurusan ilmu komputer. Kesibukan saya sehari-hari adalah menjalani perkuliahan, membangun sebuah bisnis pada bidang budidaya pertanian yaitu hidroponik, tetapi terjeda karena saya sudah berada di kampus sedangkan instalasi hidroponik saya berada di rumah.

Saya sebelumnya bersekolah di SMA Negeri 2 Pati. Sekolah yang menurut saya sangat disiplin sekali dalam hal waktu. Ketika saya masih di SMA, saya mengikuti organisasi ROHIS (Kerohanian Islam). Di dalam organisasi tersebut, saya tidak hanya belajar tentang agama islam itu sendiri. Selain itu, saya juga mengikuti kegiatan ekstrakurikuler karya ilmiah. Dalam kegiatan tersebut saya mendapatkan banyak sekali teman yang hebat-hebat.

Selanjutnya, setelah saya lulus dari SMA, saya memutuskan untuk berkuliah di Universitas Negeri Semarang dengan program studi Teknik Informatika. Saya tertarik sekali ingin menekuni jurusan ini. Saya sadar bahwa banyak sekali bidang yang ada di dalamnya, Oleh karena itu, saya ingin memfokuskan diri saya terlebih dahulu dalam bidang web. Jika saya merasa sudah mengusai bidang ini, maka saya akan merambah pada bidang lain.
